SECTION ONE
Executive Summary

A specification tool that
does the engineering for you.

The SuDS Enviro Configurator turns the deep technical knowledge required to specify a drainage chamber into a guided, ten-step experience that anyone in the supply chain can confidently complete in minutes.

The platform combines a real-time rule engine, a live three-dimensional preview and an automated drawing and specification output. Every selection is validated against physical constraints and adoption standards before the user can proceed, so the chamber that is quoted is the chamber that can be manufactured.

The Rule Engine
SECTION THREE
Manufacturing Constraints, Encoded

Seven rules that
guarantee buildability.

The rule engine runs client-side and applies in real time. The user cannot move forward with a configuration that breaches a physical or regulatory constraint.

R1
Inlet Limits
Maximum inlet count is determined by chamber diameter. 450mm allows 2 inlets, 1050mm allows 6.
R2
Outlet Sizing
Three or more inlets on a 600mm chamber, or four on any chamber, locks the outlet to a minimum of 225mm twinwall.
R3
Blocked Positions
A 225mm twinwall outlet physically blocks the 5 and 7 o'clock inlet positions adjacent to the outlet stub.
R4
Maximum Depth
Adoptable chambers are capped at 2000mm under SfA7. Non-adoptable chambers may go to 3000mm.
R5
Outlet Position
The outlet is fixed at 6 o'clock for every chamber. Displayed as a locked node on the clock face.
R6
Flow Increase
Inlet pipes cannot exceed the outlet pipe size. Hydraulic flow can never increase on exit from the chamber.
R7
Pipe Size Cap
Maximum inlet pipe size is determined by chamber diameter. 450mm caps at 160mm; 1050mm allows up to 450mm.

SECTION FOUR
What the Platform Does

Built for the whole
order journey.

The configurator is more than a wizard. It is a full operations platform that captures enquiries, generates engineering documents and gives the SuDS Enviro team a single place to manage every quote.

Specifier-facing

  • Guided ten-step configuration wizard
  • Real-time 3D chamber preview
  • Live rule validation with inline warnings
  • Auto-saved configurations with shareable URLs
  • Two-page engineering specification PDF
  • Enquiry form with instant email confirmation
  • Mobile-first responsive design

SuDS Enviro Team

  • Password-gated admin portal
  • Enquiry inbox with status workflow
  • Saved configuration browser
  • Quote dashboard with send-and-track
  • Resend-powered transactional email
  • Reviewer feedback tool with pin-drop comments
  • Full configuration audit trail
Highlight
Live Preview
Three-Dimensional Chamber Viewer
A floating button reveals a full-screen 3D model that updates as inlets, depth and pipe sizes are selected. The user can rotate, pinch and inspect from any angle.
Document Output
Specification + Drawing PDF
A single PDF combines the technical specification table, compliance check and a scale engineering drawing rendered mathematically from the configuration.
