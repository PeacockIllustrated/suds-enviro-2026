SECTION ONE
What the admin portal is for

A single back office for
every customer touchpoint.

The admin portal sits behind /admin and is the operational home for every enquiry, configuration, quote and feedback comment that flows through the SuDS Enviro Configurator.

Manage
Six Workspaces
Dashboard, Enquiries, Configurations, Quotes, Feedback and Settings - each tailored to a single job and built around table, panel and modal patterns the team already knows.
Convert
Enquiry to Quote in Minutes
A new enquiry lands with the customer's full configuration attached. Open it, draft the quote inside the portal, send the PDF, and watch the status flip as the customer interacts.
Improve
Built-in Feedback Loop
The reviewer tool at /review lets your team drop pinned comments anywhere in the live app. Every comment lands here with status, priority, and dev notes for triage.
Secure
Single Password Gate
A single ADMIN_PASSWORD environment variable protects the portal. Sessions last 24 hours and are httpOnly cookies. No accounts to manage, no users to onboard.

Configurations
/admin/configurations
Every spec a customer has built

Browse the full library of
customer configurations.

Every wizard journey - finished or in-progress - is auto-saved. The configurations workspace is the searchable archive of everything customers have built, ready to revisit, duplicate, or quote.

List & grid views

  • Toggle between a card grid (3 wide on desktop) and a dense list view.
  • Filter by product, status (draft / submitted / complete) and free-text product code.
  • Each card / row shows product, generated code, status and creation date.
  • Pagination at 18 per page; reset filters in a single click.

Detail side panel

  • Full wizard state - every selection the customer made, in plain English.
  • Generated product code and matching brochure shortcut.
  • Direct link to download the engineering drawing PDF.
  • Shareable URL the customer used so the team can replay the journey.

Settings & Access
/admin/settings
Account, system and housekeeping

Quiet controls in the
background.

Account

  • Update the admin password (current password required, 6-character minimum).
  • Inline validation, success and error messaging.

System info

  • App version and Supabase connection indicator (live colour status).
  • Total configuration and enquiry counts.
  • Last backup timestamp.

Danger zone

  • Archive draft configurations older than 30 days.
  • Confirmation modal shows the exact count that will be archived.
  • Reversible only via the database - guarded by a red call-to-action.

Authentication

  • Single password gate driven by the ADMIN_PASSWORD environment variable.
  • HTTP-only session cookie, 24 hour expiry.
  • No user accounts to manage - one shared password.
Day-in-the-life flow
1
Sign in at /admin - one shared password gives the whole team access.
2
Land on the dashboard - scan the headline stats and the recent enquiry / configuration tables.
3
Open a new enquiry - read the customer's message and the auto-attached configuration in one panel.
4
Click "New Quote" - the quote builder opens pre-filled with customer data; add line items, save and send.
5
Triage feedback in the Feedback workspace - move pins from New through to Resolved as the dev team works through them.
